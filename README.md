# Manual QA Portfolio

This repository contains hands-on **Manual Quality Assurance work** created as part of a structured QA training and practice program.

The goal of this repository is to demonstrate:

- understanding of **manual QA fundamentals**
- ability to design **test cases and test plans**
- ability to identify and document **defects clearly**
- experience with **UI and API testing**
- structured, professional QA documentation
