# Checkout test plan

## Scope

Testing the checkout functionality of a demo e-commerce application.

## In scope

- Access to checkout
- Required field validation
- Basic happy path checkout
- Input validation and edge cases

## Out of scope

- Payment gateway integration
- Performance/load testing
- Security testing

## Test types

- Functional testing
- Negative testing
- Exploratory testing

## Entry criteria

- Application is accessible
- Test environment is stable

## Exit criteria

- All planned test cases executed
- Critical defects reported
