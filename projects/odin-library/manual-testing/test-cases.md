### TC-LIB-001 – Add book with valid data

- Preconditions: Library app loaded, no modal open.
- Steps:
  1. Click "Add Book".
  2. Enter valid title, author, year, pages.
  3. Set "Read" checkbox.
  4. Click "Save".
- Expected:
  - Modal closes.
  - New row appears in the table with correct data.
  - Counters update correctly (total/read/unread).
